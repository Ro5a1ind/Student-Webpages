My Projects

Stopwatch

This stopwatch app lets you start, stop, and reset time easily. It is built with a responsive digital timer made with HTML, CSS, and JavaScript. This project demonstrates a solid understanding of time-based JavaScript logic, conditional formatting, and DOM manipulation. Custom styling adds personality while maintaining usability.

Typing Game

This Typing Game is an interactive web-based game designed to improve typing speed and accuracy. Players select a difficulty level and are then tasked with typing sentences as accurately as possible to earn points. The game features real-time feedback on typing accuracy, a scoring system, and a timer for higher difficulty levels.

Guess the Number

This interactive JavaScript-based game challenges players to guess a randomly generated number. The player can choose from three difficulty levels: easy, medium, and hard. With real-time feedback, players receive hints if their guess is too high or too low. The number of attempts is tracked and displayed. The game restarts automatically when a correct guess is made or attempts run out. This project helped reinforce logic-based programming, event handling, and DOM manipulation in JavaScript while also integrating user-friendly UI design.

Notes App

Organize your thoughts and ideas with this simple notes app. Built with HTML, CSS, and JavaScript, the app lets users:Create editable notes, save notes automatically using local storage, delete notes with a simple click, search notes instantly with a built-in live filter. This project highlights how simple JavaScript functions like localStorage, and addEventListener. The added search feature was an original enhancement beyond the tutorial, showing initiative in customizing and improving functionality.

Random Password Generator

This is a colorful and interactive password generator built with HTML, CSS, and JavaScript. The app creates two strong, randomly generated passwords side by side using a mix of uppercase letters, lowercase letters, numbers, and special symbols. With a single click of the "Generate Password" button, users instantly receive secure passwords that are ideal for protecting online accounts. This project taught the importance of secure password practices, and randomization using arrays.
